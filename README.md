# shuffle

This tool is useful to shuffle the lines in an input file.

## How to use

```sh
$ cat list.txt
Masahiro Nakai
Takuya Kimura
Goro Inagaki
Tsuyoshi Kusanagi
Shingo Katori

$ ./shuffle_lines.py list.txt 
1: Masahiro Nakai
2: Shingo Katori
3: Tsuyoshi Kusanagi
4: Takuya Kimura
5: Goro Inagaki
```
